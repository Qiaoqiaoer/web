    HTML是什么，HTML5是什么
  • HTML5是什么 HTML 指的是超文本标记语言 (Hyper Text Markup Language),而H5是HTML标准规范的第五次修改版。
    HTML元素标签、属性都是什么概念？
  • HTML元素标签是用来解释标签内的内容从而解释和构建页面的内容的手段。属性是为元素标签提供附加信息的手段。
    文档类型是什么概念，起什么作用？
  • 文档类型即常见的 !DOCTYPE ,是用来声明并帮助浏览器正确识别网页。因为HTML发展至今有许多的不同版本，所以声明不同的文档类型可以让浏览器完全正确地显示出 HTML 页面。

    表单标签都有哪些，对应着什么功能，都有哪些属性
  • form标签
  • 定义一个 URL。当点击提交按钮时，向这个 URL 发送数据。
  • data 供自动插入数据。
  • replace 定义表单提交时所做的事情。
  • accept 处理该表单的服务器可正确处理的内容类型列表（用逗号分隔）。
  • accept-charset 表单数据的可能的字符集列表（逗号分隔）。默认值是 "unknown"。
  • enctype 用于对表单内容进行编码的 MIME 类型。
  • method 用于向 action URL 发送数据的 HTTP 方法。默认是 get。
  • name 为表单定义一个唯一的名称。不支持。用 id 代替。
  • target 在何处打开目标 URL。
